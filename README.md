# Open Source Accepted Proposal Archive

A public collection of **accepted open-source proposals** to help future applicants learn how successful proposals are written.

This repository aims to help contributors understand **how successful proposals are structured, written, and presented**. By making these proposals publicly accessible, we hope to help future applicants learn from real examples and improve their chances of getting selected.

## Repository Structure

Proposals are organized by **program and year**.

```
proposals
│
├── gsoc
│   ├── 2024
│   ├── 2025
│   └── 2026
│
├── lfx
│   ├── 2025
│   └── 2026
│
└── outreachy
```

Each proposal folder typically contains:

```
project-name
│
├── proposal.pdf
└── metadata.md
```

Example:

```
proposals/gsoc/2025/CircuitVerse-flutter-upgrade/
│
├── proposal.md
└── metadata.md
```

## Purpose

Writing a strong proposal is one of the hardest parts of applying to open-source programs. Many applicants struggle because there are **very few public examples of accepted proposals**.

This repository exists to:

- Provide **real examples of successful proposals**
- Help applicants understand **structure, clarity, and technical depth**
- Serve as a **learning resource for open-source contributors**
- Create a **public archive of open-source proposals**

---

## Programs Covered

Currently, the repository includes proposals from:

- **Google Summer of Code (GSoC)**
- **LFX Mentorship**
- **Summer of Bitcoin**
- **Code for Good Tech (C4GT)**
- **Outreachy**

## Contributing

We welcome contributions from anyone who would like to share their accepted proposal.

### Steps to Contribute

1. Fork this repository.
2. Add your proposal under the appropriate program and year.
3. Create a folder for your project:

```
proposals/<program>/<year>/<org-name-project-name>/
```

4. Add the following files:

- `proposal.pdf`
- `metadata.md`

5. Submit a Pull Request.

---

## Metadata Format

Please include a `metadata.md` file with the following details:

```
Author: Your Name
College/Organization: Your University or Organization
Program: GSoC / LFX / Outreachy
Year: 2026
Project: Project Name
Mentor Organization: Organization Name
```
